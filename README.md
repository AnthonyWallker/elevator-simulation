# 🚀 Elevator Simulation

Interactive elevator simulation built with:

- TypeScript
- PIXI.js
- Tween.js
- Vite

## ✨ Features

### Core functionality
- Configurable floors count
- Configurable elevator capacity
- Passenger spawn system
- Elevator movement simulation
- Boarding and exiting flow
- Direction-based passenger logic

### Animations
- Smooth elevator movement
- Passenger walking animations
- Boarding animations
- Exit animations

### Bonus task
- Smart elevator routing
- No unnecessary stops
- Direction-aware movement system

---

## 🧠 Architecture

Project structure:

```text
src/
├── config/
├── core/
├── entities/
├── render/
├── systems/
└── main.ts
```

### Main concepts
- Entity-based architecture
- Finite State Machine (FSM)
- Simulation loop
- Rendering layer separation
- Tween animation pipeline

## 🎮 Controls / Configuration

You can configure:

```ts
floors: 4-10

```

Inside:

```text

```
