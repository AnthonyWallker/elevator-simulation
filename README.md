# Elevator Simulation

Test assignment implemented with:

- TypeScript
- PIXI.js
- Tween.js
- Vite

## Features

- Configurable floors count
- Configurable elevator capacity
- Passenger spawn system
- Elevator movement simulation
- Boarding and exiting animations
- Smart routing system

## Run project

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
